# ODM 首轮产品与价格等级调整设计

日期：2026-07-13

## 目标

更新 Memova ODM 合作介绍页，使首轮合作与验证范围明确包含录音卡、夹耳式耳机和智能眼镜，并移除产品矩阵中的具体美元售价，以相对价格等级表达产品定位。

## 首轮合作与验证范围

首轮合作产品统一为：

1. 录音卡
2. 夹耳式耳机
3. 智能眼镜

页面中所有涉及首发路径、第一单合作包、Phase 1、首轮目标、验证重点、阶段产出和 ODM 沟通问题的文案都应同步反映这三种产品，避免只修改单一区块造成前后矛盾。

产品验证重点分别为：

- 录音卡：设备激活、音频捕捉、App 上传、Memory OS 结构化和 Agent 行动闭环。
- 夹耳式耳机：全天候佩戴、听说交互、私人提示、续航和声学表现。
- 智能眼镜：第一视角视觉上下文、音视频采集、佩戴体验、隐私指示和端到端数据链路。

三种产品均属于首轮合作与验证范围，但具体工程并行度、打样顺序和平台复用方案由双方根据 ODM 的现有 reference design 在工程评估阶段确定。

## 产品价格等级

删除所有面向消费者的具体美元售价，替换为以下模糊价格等级：

| 产品 | 价格等级 |
| --- | --- |
| 录音卡 | 入门级 |
| 手表扣 | 主流级 |
| 夹耳式耳机 | 主流级 |
| 智能眼镜 | 专业级 |
| Memova Pen | 奢侈品级 |

价格等级仅表达产品组合中的相对定位，不构成建议零售价、目标 BOM 或量产报价。原“消费者建议零售价方向”说明同步改为“产品价格等级为方向性定位”。

## 页面范围

本次仅修改隐藏页面 `/odmpartnership/` 及其相关自动化测试，不调整主页、团队成员页、SEO 索引策略或其他公开页面。

页面继续保持 `noindex, nofollow`。

## 验收标准

- 页面不再出现 `US$90`、`US$120`、`US$150`、`US$200` 或 `US$500`。
- 五种硬件分别显示已确认的价格等级。
- 录音卡、夹耳式耳机、智能眼镜在所有首轮合作与验证文案中保持一致。
- 桌面端和 390px 宽移动端无横向溢出，价格等级不会破坏卡片布局。
- TypeScript 检查、生产构建和相关 SEO 静态页面测试通过。
- 本地修改完成后不自动部署；发布到 Cloudflare Pages 前另行获得用户明确确认。
