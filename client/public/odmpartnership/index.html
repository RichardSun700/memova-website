08 / Execution Gates

用四个 gate 控制投入，而不是一次性押注量产

ODM 内部立项需要明确：每一阶段花什么钱、解决什么风险、产出什么证据、什么条件触发下一阶段。Memova 建议先用工程 gate 管理合作，再讨论规模订单。

以下是建议验证窗口，不是无条件交付或采购承诺。 90-180 天路径以采用成熟 reference design、首轮不进行大规模定制开模或架构重置、核心器件供应稳定，以及双方及时冻结接口与验收标准为前提。首批设备用于样机、用户测试、渠道验证、认证准备和商业化决策。
Gate 1｜设计评估30 天锁定首轮组合三种产品的 reference design、BOM、接口、结构、认证与样机周期
Gate 2｜样机闭环90 天端到端演示音视频捕捉、App 激活、Memory OS、Agent follow-up
Gate 3｜工程验证150 天EVT / DVT 决策声学、续航、连接、隐私指示、数据完整性和测试标准
Gate 4｜试产计划180 天小批量方案试产报价、首批数量、NRE 抵扣、第二 SKU 复用范围
